# Service types
### Compute
###### Amazon Elastic Compute Cloud (EC2)
- is a web service that provides resizable computing capacity—literally, servers in Amazon's data centers—that you use to build and host your software systems.
- can use with Auto Scaling by create collections of EC2 instances as Auto Scaling groups.
###### AWS Elastic Beanstalk
- you can quickly deploy and manage applications in the AWS Cloud without worrying about the infrastructure that runs those applications.
- reduces management complexity without restricting choice or control. 
- You simply upload your application, and AWS Elastic Beanstalk automatically handles the details of capacity provisioning, load balancing, scaling, and application health monitoring.
